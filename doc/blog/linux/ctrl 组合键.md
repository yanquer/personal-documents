### ctrl 组合键

- ctrl + c    终止目前的命令
- ctrl + d    输入结束（EOF），例如邮件结束的时候
- ctrl + M    enter
- ctrl + s    暂停屏幕输出
- ctrl + q    恢复屏幕输出
- ctrl + u    在提示字符下，将整列命令删除
- ctrl + z    暂停 目前的命令

当输入一串指令时，发现前面写的一串数据是错的，想要删除游标所在处到最前面的指令内容，应该如何处理？

答：ctrl + u

如何得知目前所有的变量与环境变量

环境变量：env或export

所有变量：set

注意 变量不能以数字开头

若 A=B 且 B=C，若我下达 unset $A，则取消的是什么？

取消的是B，因为 unset $A 相当于 unsetB 所以取消的是B，A会继续存在
